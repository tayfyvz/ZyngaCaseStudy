Game Developer Test for Zynga - Jewel object matching game on a grid
	

	This game is developed in an event driven perspective. To enable event driven architecture,
EventDrivenFramework is used. This framework enables communication between managers which inherits
BaseManager abstract class. Event packages are reveived in Pub-Sub style mechanism (Observer Pattern).
